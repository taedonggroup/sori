# 디자인 검수 스킬

## 검수 기준 (2026 UI/UX 트렌드)
- [ ] 다크/라이트 모드 대응
- [ ] 모바일 퍼스트 반응형 (375px, 768px, 1280px)
- [ ] 로딩 스켈레톤 UI 적용
- [ ] 마이크로 인터랙션 (호버, 클릭 피드백)
- [ ] 접근성: aria-label, 색상 대비 4.5:1 이상
- [ ] shadcn/ui 컴포넌트 일관성
- [ ] 한국어 폰트 최적화 (Pretendard 또는 Noto Sans KR)

## 공간(메인) 검수 포인트
- 원석을 감성적으로 맞이하는 첫 화면인가
- 조각 업로드 CTA가 명확한가
- 공명 브랜드 정체성이 느껴지는가

## 검수 결과 형식
통과 / 수정 필요 / 재작업 필요 3단계로 분류하고
수정 필요 항목은 구체적인 코드 수정 방향을 제시.
